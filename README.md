# PhotoAI: AI-Powered Image and Video Editing

**PhotoAI** is a modern web-based platform that utilizes **Cloudinary's AI API** to provide advanced image and video editing tools. Designed for creators and developers, PhotoAI offers seamless AI-powered features, ensuring an effortless editing experience.

---

## Features

### AI-Powered Tools

- **Image Editing**

  - Recolor images with AI-enhanced vibrancy.
  - Smart cropping for professional framing.
  - Export images directly from video content.

- **Video Editing**

  - Transcribe audio to text for captions or analysis.
  - Resizeable video editor for dynamic content adjustments.
  - Smart cropping to focus on key elements.

- **Upload and Manage Media**

  - Intuitive drag-and-drop media uploader.
  - Cloudinary-powered backend for fast and secure processing.

- **Responsive Design**
  - Built with modern UI components for a smooth user experience.

### Animations and Visuals

- Engaging animations for AI processing, cloud operations, and video uploads.
- Sleek visual elements for improved interactivity.

---

## Usage

1. Upload Media
   Drag and drop your images or videos into the upload tool for editing.

2. Edit Media
   Use the AI-powered toolbar to:

   - Recolor, crop, or resize iamges.
   - Transcribe video audio to text.
   - Apply smart cropping to videos.

3. Export and Download
   Save and download your processed files directly.

## Technologies Used

### Frontend

- **React**: For building a responsive and dynamic user interface.
- **Next.js**: To enable server-side rendering and efficient routing.
- **ShadcnUI**: For modern, accessible UI components.
- **Tailwind CSS**: To style the application effortlessly.

### Backend

- **Cloudinary**: For AI-powered image and video processing.
- **Zod**: To validate server-side data.
- **Zustand**: For efficient state management.

### Animations

- **Lottie-React**: For rendering smooth animations.
